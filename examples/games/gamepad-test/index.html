ブラウザの Gamepad / Keyboard API をそのまま読むだけの診断ページ（SDK・helper 不要）。コントローラやキーが「検出されるか」「どのボタンが取れるか」を確認できます。

反応しない時のチェック

  • このページをクリックしてフォーカスし、その状態でコントローラのボタンを1回押す（ブラウザはボタン押下まで検出しません）。
  • 複数の「パッド」が検出されることがあります（USB スピーカーフォン等が誤検出される）。上のプルダウンで本物のコントローラ（STANDARD GAMEPAD）を選べます。ゲーム側は標準パッドを自動優先します。
  • Bluetooth 自体は問題ではありません。USB でも BT でも動きます。まず OS 側でコントローラがペアリング/認識されているか確認。
  • ブラウザは Chrome / Edge 推奨。Firefox はボタン番号(mapping)がずれることがあります。
  • 「状態」に id が出て mapping=standard なら検出 OK。ボタンが点灯すればブラウザ側は正常です。
